Hexo + github 博客搭建过程

最近想建个博客记录一下平时的一些技术细节，以后可以随时翻翻，发现以前做过的事总是很容易忘记，真是好记性不如烂笔头,顺便整理一下思路。

  • 整理知识，学习笔记
  • 发布日记，杂文，所见所想
  • 撰写发布技术文稿（代码支持）
  • 撰写发布学术论文（LaTeX 公式支持）

搭建环境

  • Windows10 64位
  • node@4.4 后变成6.1.0
